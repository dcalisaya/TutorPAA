# Roadmap

Status: draft  
Purpose: ordenar el crecimiento del proyecto sin volverlo pesado.

## Fase 0: Fundacion CLI

- Estructura del repo.
- Contenido semilla en JSONL.
- CLI sin dependencias externas como herramienta para agentes.
- Prompts iniciales para agentes.
- Perfil local de estudiante.
- Flujo agent-first documentado.

## Fase 1: Preparacion Real

- Diagnostico por estudiante.
- Practica adaptativa basica.
- Registro de intentos.
- Reporte por habilidades.
- Simulacro corto.
- Mas preguntas curadas.
- Plan de estudio guiado con las fuentes oficiales.

## Fase 2: Motor Local

- SQLite para sesiones.
- Validador de contenido.
- Scoring mas estable.
- Import/export de progreso.
- Revision asistida de preguntas generadas por AI.
- Soporte opcional para redaccion indirecta e ingles si se confirma su necesidad.

## Fase 3: React + Vite

- App web con React + Vite + TypeScript.
- Dashboard de progreso.
- Practica visual.
- Simulacros cronometrados.
- Editor de preguntas.

## Fase 4: Comunidad

- Guia de contribucion.
- Banco amplio de preguntas.
- Revisores por area.
- Publicacion de releases de contenido.
