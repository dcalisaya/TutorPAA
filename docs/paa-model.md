# Modelo PAA Para TutorPAA

Status: draft  
Purpose: definir el modelo educativo inicial usado por el repo.

## Alcance

TutorPAA prepara para aptitud verbal y aptitud matematica con un modelo interno de practica sobre 2400 puntos:

| Seccion | Puntaje modelado |
|---|---:|
| Aptitud Verbal | 1200 |
| Aptitud Matematica | 1200 |
| Total | 2400 |

Este modelo es una abstraccion de preparacion. Las reglas oficiales de admision, fechas, costos y requisitos deben validarse siempre con la USFQ.

## Fuentes Oficiales

Las fuentes oficiales iniciales estan listadas en `docs/official-sources.md`.

La guia de estudios PAA de College Board describe una prueba mas amplia con Lectura, Redaccion, Matematicas e Ingles. Para el objetivo USFQ inicial, TutorPAA prioriza:

- Verbal: lectura, vocabulario en contexto, inferencias, analisis textual y redaccion indirecta como complemento.
- Matematica: aritmetica, algebra, geometria, analisis de datos y probabilidad.

Ingles queda como area complementaria para ubicacion o preparacion adicional hasta confirmar su peso operativo en el proceso especifico del estudiante.

## Habilidades Iniciales

Matematica:

- aritmetica
- algebra
- geometria
- funciones
- razonamiento cuantitativo
- interpretacion de datos

Verbal:

- comprension lectora
- vocabulario en contexto
- inferencia
- estructura del texto
- completacion de oraciones
- razonamiento verbal
- redaccion indirecta

Complementario:

- uso del ingles
- comprension de lectura en ingles
- redaccion indirecta en ingles

## Diagnostico

El diagnostico inicial debe medir:

- precision por seccion
- tiempo aproximado por pregunta
- temas debiles
- tipo de error dominante
- confianza reportada por el estudiante

## Clasificacion De Errores

Usar estas categorias:

- `concept`: no domina el concepto.
- `calculation`: entiende, pero falla en calculo.
- `reading`: leyo mal datos o pregunta.
- `strategy`: eligio un metodo lento o fragil.
- `time`: no alcanzo a resolver.
- `careless`: descuido puntual.

## Puntaje Estimado

El puntaje estimado inicial debe presentarse como orientativo, no oficial. Mientras haya pocas preguntas, usar rangos amplios y explicar que la muestra es pequena.
