# Web App Futuro

Status: planned  
Stack previsto: React + Vite + TypeScript.

La web sera una capa visual sobre el mismo contenido y progreso usado por la CLI.

Objetivos:

- Dashboard de progreso.
- Practica interactiva.
- Simulacros cronometrados.
- Revision de errores.
- Editor de preguntas con validacion.

No iniciar aqui hasta que el flujo CLI y el modelo de contenido esten validados.
